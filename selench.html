selench.driver module

Main module that holds all the methods to interact with the browser.

from selench import Selench, Keys

driver = Selench()
driver.get('https://google.com')
driver.element('input[name="q"]').send_keys('Hello World!', Keys.ENTER)
driver.quit()
class selench.driver.Selench(browser: Literal['chrome', 'firefox'] = 'chrome', wait: int = 10, executable_path: str = None, headless: bool = False, user_agent: str = None, incognito: bool = False, binary_location: str = None)[source]

This is the main class that holds all the methods to interact with the browser.

Parameters:

  • browser – The browser to use.

  • wait – The default explicit wait time for WebDriverWait.

  • executable_path – The path to the driver executable.

  • headless – Run browser in headless mode.

  • user_agent – The user agent to use.

  • incognito – Run browser in incognito mode.

  • binary_location – The location of the browser executable.

select_element(element_or_selector: str | WebElement) → Select[source]

Convert a web element to a Select object. The Select object provides a convenient way to interact with select elements (drop-down lists) on a webpage. It allows the user to select one or more options from the list, and also provides methods to retrieve the selected options, as well as other useful information about the select element.

Parameters:

element_or_selector – The web element or selector to convert to a Select object.

Returns:

The converted Select object.

Return type:

Select

Example:

element = driver.css_element('select')
select_object = driver.select_element(element)

# Select an <option> based upon the <select> element's internal index
select_object.select_by_index(1)

# Select an <option> based upon its value attribute
select_object.select_by_value('value1')

# Select an <option> based upon its text
select_object.select_by_visible_text('Bread')
